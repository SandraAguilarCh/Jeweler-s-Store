# Jeweler's Store - Tienda Online de Accesorios
# Descripción del proyecto:
Este proyecto consiste en la creación de un sitio web para una tienda online para dar a conocer los accesorios
de alta calidad, con el objetivo de ofrecer una experiencia intuitiva y funcional que facilite la navegación y
la gestión del proceso de comprar desde la selección del carrito.

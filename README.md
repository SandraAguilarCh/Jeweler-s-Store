# Jeweler's Store - Tienda Online de Accesorios
# Descripción del proyecto:
Este proyecto consiste en la creación de un sitio web para una tienda online para dar a conocer los accesorios
de alta calidad, con el objetivo de ofrecer una experiencia intuitiva y funcional que facilite la navegación y
la gestión del proceso de comprar desde la selección del carrito.

# Características Destacadas
-Navegación por Categorías: El catálogo está claramente clasificado, permitiendo al usuario acceder fácilmente a cada tipo de accesorio.    
-Detalle de Productos: Cada artículo presenta su nombre, precio unitario e imagen descriptiva.                             
-Gestión de Carrito de Compras: Funcionalidad completa para agregar productos al carrito, modificar cantidades y eliminar ítems antes de la compra final.                               
-Filtros de Búsqueda Avanzados: Permite encontrar productos por tipo, rango de precios o materiales.                   
-Autenticación de Usuarios: Los clientes pueden registrarse e iniciar sesión para guardar su historial y preferencias.
